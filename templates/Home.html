Bangla News Headline Generator

What is this ?

This is an web application to generate Bangla news headline providing a news article as input.

How it works?

The backend server contains a trained deep learning model that generates the headlines by consuming news articles.

Which technologies are used?

PyTorch, Tensorflow, TorchText and other libraries for model creation and Flask for model deployment.

Who developed this project? 

The project is funded by ICT division and developed my a team of meachine learning experts.
